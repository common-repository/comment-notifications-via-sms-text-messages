== Description ==

This plugin will send comment notifications via SMS / text messages. This is powered by the txtweb.com APIs.

To set it up, 
SMS "@comment" (without quotes) to 
"9243342000" [India] 
"650 385 8882" or "898932" [US or Canada] 

to acquire this Identifier code.

Go to Wordpress Dashboard --> Settings --> Comment Notifications via SMS and set the code.

PS: If you have any other ideas for an SMS based plugin and want to share it, [contact me](http://suhastech.com/)

== Installation ==

1. Unzip the package, and upload it to the "/wp-content/plugins/" directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it. 


== Frequently Asked Questions ==

= I don't recieve SMS notfications. What do I do? =
The Identifer Code might be entered wrong. Please try sending "@comment" again.
